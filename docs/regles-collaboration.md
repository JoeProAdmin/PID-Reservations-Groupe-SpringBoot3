# Regles de collaboration - PID Reservations Groupe

## Dépôt officiel du groupe
Le travail de groupe doit être réalisé uniquement sur le dépôt :

PID-Reservations-Groupe-SpringBoot3

## Important
Le dépôt individuel de Youssef ne doit pas être utilisé pour le travail collaboratif du groupe.

## Organisation technique
- backend : dossier `backend`
- documentation : dossier `docs`
- les futures contributions frontend seront ajoutées dans un dossier dédié

## Règles de contribution
1. Chaque membre doit prévenir avant de commencer une partie importante.
2. Les modifications doivent être claires et limitées.
3. Les messages de commit doivent être compréhensibles.
4. Le dépôt doit rester propre.
5. Les fichiers `.idea`, `target` et autres fichiers locaux ne doivent pas être envoyés.

## Intégration
Le responsable de l’intégration valide la cohérence finale avant les dépôts importants.
